## 概要

PDFハンドラは、ファイルサーバ上のPDFファイルを効率的に管理・編集するためのデスクトップアプリケーションです。

### 主な機能

- ✅ PDFファイルのプレビュー表示
- ✅ サムネイル一覧表示
- ✅ ファイル名の変更（F2キー）
- ✅ 複数PDFの結合
- ✅ PDFの分割
- ✅ お気に入りフォルダ管理
- ✅ ファイルの削除

---

## 起動方法

### 初回起動

1. デスクトップのショートカットをダブルクリック
2. または、スタートメニューから「PDFハンドラ」を選択

### フォルダを開く

1. **ツールバー**の「📁 開く」ボタンをクリック
2. または、**メニューバー**の「ファイル → フォルダを開く」を選択
3. PDFファイルが保存されているフォルダを選択

## 基本操作

### フォルダを選択

**方法1: ツリーから選択**
1. 左ペインのフォルダツリーでフォルダをクリック
2. 中央ペインにPDFファイル一覧が表示される

**方法2: お気に入りから選択**
1. ツリー上部の「⭐お気に入り」を展開
2. 登録済みフォルダをクリック

### ファイルを選択

**単一選択:**
- ファイルをクリック

**複数選択:**
- `Ctrl` + クリックで複数選択
- `Shift` + クリックで範囲選択

### ファイルを開く

**方法1: ダブルクリック**
- サムネイルまたはリスト上でファイルをダブルクリック
- デフォルトのPDFリーダーで開きます

**方法2: Enter キー**
- ファイルを選択して `Enter` キーを押す

## ファイル操作

### ファイル名を変更

**F2キーで変更（推奨）:**

1. ファイルを選択
2. `F2` キーを押す
3. ファイル名を編集
4. `Enter` で確定（`Esc` でキャンセル）

**注意:**
- `.pdf` 拡張子は自動的に付きます
- 使用できない文字: `\ / : * ? " < > |`

### ファイルを削除

**手順:**
1. ファイルを選択
2. `Delete` キーを押す（または🗑️ボタン）
3. 確認ダイアログで「はい」をクリック

**警告:**
- 削除したファイルは元に戻せません
- 慎重に操作してください

### ファイルをコピー

1. ファイルを選択
2. `Ctrl` + `C` でコピー
3. エクスプローラーに貼り付け

## PDF操作

### PDF結合

**複数のPDFを1つに結合:**

1. 結合したいPDFファイルを複数選択（`Ctrl` + クリック）
2. ツールバーの「🔗 結合」ボタンをクリック
3. または `Ctrl` + `M`
4. 結合ダイアログが表示される

**ダイアログの操作:**
```
┌─────────────────────────────────────────┐
│  PDFファイルの結合                      │
├─────────────────────────────────────────┤
│  結合するファイル:                      │
│  ┌─────────────────────────────────┐   │
│  │ 1. report.pdf                   │ ↑ │
│  │ 2. invoice.pdf                  │ ↓ │
│  │ 3. manual.pdf                   │   │
│  └─────────────────────────────────┘   │
│  [ファイル追加] [削除] [上へ] [下へ]   │
│                                         │
│  保存先: C:\Output\                     │
│  ファイル名: merged.pdf                 │
│                                         │
│           [結合実行] [キャンセル]       │
└─────────────────────────────────────────┘
```

**操作:**
- **上へ/下へ**: 結合順序を変更
- **ファイル追加**: PDFを追加
- **削除**: 選択したファイルを除外
- **結合実行**: 結合を開始

### PDF分割

**1つのPDFを複数に分割:**

1. 分割したいPDFファイルを選択
2. ツールバーの「✂️ 分割」ボタンをクリック
3. または `Ctrl` + `Shift` + `S`
4. 分割ダイアログが表示される

**分割方法の選択:**

**① ページ範囲で分割**
```
範囲: [1-3], [4-7], [8-10]
→ 3つのPDFに分割
```

**② 1ページずつ分割**
```
10ページのPDF → 10個のPDFファイル
```

**③ 等分割**
```
分割数: 3
→ ページ数を3等分
```

## 表示設定

### サムネイルビュー ⇔ リストビュー

**切り替え方法:**
- ツールバーの「🖼️ サムネイル」ボタンをクリック
- または `メニュー → 表示 → サムネイルビュー/リストビュー`

**サムネイルビュー:**
- PDFの第1ページを画像で表示
- 視覚的に内容を確認できる

**リストビュー:**
- ファイル名、サイズ、更新日時を表示
- 詳細情報を一覧で確認

### サムネイルサイズの変更

**手順:**
1. メニューバー → 表示 → サムネイルサイズ
2. **小/中/大** から選択

**または:**
- ツールバーのスライダーで調整

### プレビュー表示 ON/OFF

**切り替え方法:**
- ツールバーの「👁️ プレビュー」ボタンをクリック
- または `F12` キー

**OFFにすると:**
- 中央ペインが広くなる
- より多くのファイルを表示できる

### ページめくり（プレビュー）

**次ページ:** `→` キー または ▶️ボタン  
**前ページ:** `←` キー または ◀️ボタン

**ズーム:**
- `Ctrl` + マウスホイール
- ズームボタン（🔍+ / 🔍-）

## お気に入り機能

### お気に入りに追加

**手順:**
1. フォルダツリーでフォルダを右クリック
2. 「お気に入りに追加」を選択
3. 名前を入力（デフォルトはフォルダ名）
4. 「OK」をクリック

### お気に入りの名前を変更

1. お気に入りのフォルダを右クリック
2. 「名前変更」を選択
3. 新しい名前を入力
4. 「OK」をクリック

### お気に入りから削除

1. お気に入りのフォルダを右クリック
2. 「お気に入りから削除」を選択

**注意:** フォルダ自体は削除されません

## トラブルシューティング

### Q1. PDFが表示されない

**確認事項:**
- ファイルが破損していないか確認
- 別のPDFリーダーで開けるか試す
- アプリを再起動してみる

**解決策:**
```
ツールバー → 🔄 更新 をクリック
```

### Q2. ファイル名が変更できない

**原因:**
- ファイルが他のアプリで開かれている
- ファイルが読み取り専用

**解決策:**
1. PDFを閉じる
2. ファイルのプロパティで読み取り専用を解除

### Q3. 結合/分割が失敗する

**確認事項:**
- 保存先フォルダに書き込み権限があるか
- ディスクの空き容量は十分か
- ファイル名に使用できない文字が含まれていないか

### Q4. サムネイルが表示されない

**解決策:**
```
1. メニュー → 表示 → 更新（F5）
2. フォルダを開き直す
3. アプリを再起動
```

### Q5. プレビューが遅い

**対処法:**
- プレビューをOFFにする（F12）
- サムネイルサイズを「小」にする
- 他のアプリを閉じる

### Q6. エラーメッセージが表示される

**「ファイルへのアクセスが拒否されました」**
- 管理者権限で実行してください

**「PDFが破損しています」**
- 元のPDFファイルを確認してください
- 別のPDFで試してください

## よくある質問（FAQ）

### Q. 一度に結合できるPDFの数は？

A. 制限はありませんが、ファイルサイズによります。数百MBを超える場合は時間がかかります。

### Q. 分割したPDFの保存先は？

A. 分割ダイアログで指定できます。デフォルトは元のPDFと同じフォルダです。

### Q. お気に入りはどこに保存される？

A. アプリケーションのデータフォルダに自動保存されます。アンインストール時に削除されます。

### Q. PDFに注釈を付けられますか？

A. 現バージョンでは未対応です。将来のバージョンで追加予定です。

### Q. パスワード付きPDFは開けますか？

A. プレビューは表示されませんが、ファイル操作（名前変更、削除など）は可能です。
