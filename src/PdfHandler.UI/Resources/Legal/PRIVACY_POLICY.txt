PDFハンドラ プライバシーポリシー

最終更新日: 2025年12月28日
バージョン: 1.0

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PDFハンドラ（以下「本ソフトウェア」）の開発者（以下「開発者」）は、
ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

本プライバシーポリシー（以下「本ポリシー」）は、本ソフトウェアにおける
個人情報の取り扱いについて説明するものです。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


第1条（収集する情報）

本ソフトウェアは、以下の情報を収集しません：

1. 個人を特定できる情報
   - 氏名、住所、電話番号、メールアドレス等

2. 利用状況に関する情報
   - 操作ログ、アクセスログ等

3. デバイス情報
   - IPアドレス、デバイス識別子等

4. 位置情報

5. クッキーその他のトラッキング技術による情報


第2条（ファイルの取り扱い）

1. 本ソフトウェアは、ユーザーが選択したPDFファイルを
   ローカル環境（ユーザーのコンピューター）でのみ処理します。

2. ユーザーのファイルは、以下のとおり取り扱われます：
   (1) ファイルは外部サーバーに送信されません
   (2) ファイルはインターネット経由で転送されません
   (3) ファイルは開発者に共有されません
   (4) ファイルは第三者に提供されません

3. ファイルの内容、ファイル名、メタデータ等は、すべてユーザーの
   ローカル環境でのみ処理されます。


第3条（インターネット通信）

本ソフトウェアは、以下の場合にのみインターネット通信を行います：

1. アップデート確認機能を使用した場合
   - 通信先: GitHub API (api.github.com)
   - 送信情報: なし（リクエストのみ）
   - 取得情報: 最新バージョン情報

2. ヘルプ・サポートリンクをクリックした場合
   - 通信先: GitHub (github.com)
   - ユーザーが明示的にリンクをクリックした場合のみ

3. 上記以外のバックグラウンド通信は一切行いません。


第4条（データの保存）

1. 本ソフトウェアは、以下の設定情報をユーザーのローカル環境に
   保存する場合があります：
   - アプリケーション設定（表示モード、ウィンドウサイズ等）
   - お気に入りフォルダのパス

2. これらの情報は、すべてユーザーのコンピューター内に保存され、
   外部に送信されることはありません。

3. 保存場所：
   - Windows: %APPDATA%\PdfHandler\ または
     C:\Users\[ユーザー名]\AppData\Local\PdfHandler\


第5条（第三者サービス）

本ソフトウェアは、以下の第三者サービスを利用する場合があります：

1. GitHub
   - 目的: アップデート確認、サポート提供
   - プライバシーポリシー: https://docs.github.com/ja/site-policy/privacy-policies/github-privacy-statement

2. これらのサービスの利用は、ユーザーが明示的に操作した場合
  （アップデート確認ボタンをクリック、リンクをクリック等）のみです。


第6条（子どもの個人情報）

本ソフトウェアは、13歳未満の子どもを対象としていません。
13歳未満の方が使用する場合は、保護者の同意を得てください。


第7条（セキュリティ）

1. 本ソフトウェアは、ユーザーデータを外部に送信しないため、
   通信に関するセキュリティリスクはありません。

2. ローカルファイルのセキュリティは、ユーザーのコンピューターの
   セキュリティ設定に依存します。

3. 開発者は、以下を推奨します：
   - ウイルス対策ソフトウェアの使用
   - OSおよびセキュリティパッチの適用
   - 重要なファイルのバックアップ


第8条（ユーザーの権利）

ユーザーは、以下の権利を有します：

1. 本ソフトウェアの使用を停止し、アンインストールする権利

2. ローカルに保存された設定ファイルを削除する権利

3. 本ポリシーについて質問、懸念を表明する権利


第9条（本ポリシーの変更）

1. 開発者は、必要に応じて本ポリシーを変更することができます。

2. 本ポリシーが変更された場合、変更後のポリシーは、本ソフトウェアの
   配布ページおよびソフトウェア内で告知されます。

3. 変更後の本ポリシーは、告知された時点から効力を生じます。

4. 重要な変更がある場合は、ソフトウェア起動時に通知する場合があります。


第10条（準拠法）

本ポリシーの解釈および適用は、日本法に準拠します。

要約

• 本ソフトウェアは個人情報を収集しません
• ファイルはローカル環境でのみ処理されます
